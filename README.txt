WHAT THIS "CREATE-REACT-APP TEMPLATE" SUPPORTS:
- this is basically CreateReactApp into static files
- latest React syntaxes ('state = {}') also supported
- JS and CSS loaders, CSS modules are enabled
- feel free to install other NPM packages



WHAT THIS "CREATE-REACT-APP TEMPLATE" DOES NOT SUPPORT (or what has to be installed):
- babel polyfill
- image loaders



HOW TO USE:
- first of all, run 'npm install' to install all dependencies
- to start a development server, open terminal and execute 'npm start'
- to build this project, execute 'npm run build' => 'dist' folder will be created
- afterwards upload your 'dist' folder wherever you need
